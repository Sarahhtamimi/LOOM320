# 🌱 LOOM – Sustainable Fashion Platform

LOOM is a web-based platform that aims to promote **sustainable fashion** by raising awareness, supporting second-use clothing, and making it easier to discover ethical fashion brands—all in one place.

---

## 📌 Project Overview

The rapid growth of fast fashion has significantly contributed to environmental pollution, resource waste, and overconsumption.  
LOOM addresses this issue by turning sustainability awareness into **practical, everyday action**.

The platform combines:
- Sustainability blog education
- Second-use (pre-owned) fashion
- Discovery of sustainable brands

---

## 🎯 Project Objectives

- Increase awareness of the environmental impact of the fashion industry  
- Help users discover sustainable fashion brands easily  
- Encourage circular fashion through second-use clothing  
- Present sustainability information in a simple and engaging way  

---

## ✨ Key Features

### 👗 Sustainable Brands
- Browse sustainable fashion brands
- Search brands by name
- Sort brands by category
- View brand profiles (background, sustainability info, official website)

### ♻️ Second Use Section
- Browse pre-owned clothing items
- Add, edit, and delete listings
- Bookmark favorite items
- Encourage clothing reuse and reduce waste

### 📚 Educational Blog
- Sustainability-focused blog posts
- Environmental statistics and quick facts
- Simplified and engaging awareness content

### 🔐 User 
- User registration
- Login and logout
- Account page
- Manage listings and bookmarks

### 📚 Admin 
- Add blog 

---

## 👥 Target Users

- Students and young adults interested in fashion and sustainability  
- Users seeking eco-friendly fashion alternatives  
- Individuals who want to sell or browse second-use clothing  
- Users with basic digital literacy and web browsing experience
